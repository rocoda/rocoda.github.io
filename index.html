RoCoDA

Counterfactual Data Augmentation for Data-Efficient Robot Learning from Demonstrations

About RoCoDA

RoCoDA is a novel framework that leverages counterfactual data augmentation to improve the efficiency of robot learning from demonstrations. By exploiting the causal structure of robot-environment interactions, RoCoDA enhances policy generalization and reduces the need for extensive training data.

Key Features

  • Counterfactual data augmentation techniques for robot learning
  • Focus on causal relationships between states and actions
  • Improves generalization to unseen tasks and environments
  • Data-efficient learning from demonstrations
